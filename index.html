    Pre-request Script APIs

    Execute code before sending the request to modify headers, set variables, or perform setup.

    request.url The request URL (string)
    request.method HTTP method (GET, POST, etc.)
    request.headers Request headers object - can be modified
    request.body Request body content
    environment.get(name) Get an environment variable
    environment.set(name, value) Set an environment variable
    console.log(...) Log messages to the Scripts tab in response

    Test Script APIs

    Run tests and assertions after receiving the response to validate data and extract values.

    response.status HTTP status code (number)
    response.statusText HTTP status text (OK, Not Found, etc.)
    response.headers Response headers object
    response.body Parsed response body (JSON object)
    expect(value).toBe(expected) Assert strict equality
    expect(value).toEqual(expected) Assert deep equality
    expect(value).toBeDefined() Assert value is not undefined
    expect(value).toBeTruthy() Assert value is truthy
    expect(value).toContain(item) Assert array/string contains item
    expect(value).toBeGreaterThan(n) Assert value is greater than n
